hbase shell命令 描述
alter 修改列族（column family）模式
count 统计表中行的数量
create 创建表
describe 显示表相关的详细信息
delete 删除指定对象的值（可以为表，行，列对应的值，另外也可以指定时间戳的值）
deleteall 删除指定行的所有元素值
disable 使表无效
drop 删除表
enable 使表有效
exists 测试表是否存在
exit 退出hbase shell
get 获取行或单元（cell）的值
incr 增加指定表，行或列的值
list 列出hbase中存在的所有表
put 向指向的表单元添加值
tools 列出hbase所支持的工具
scan 通过对表的扫描来获取对用的值
status 返回hbase集群的状态信息
shutdown 关闭hbase集群（与exit不同）
truncate 重新创建指定表
version 返回hbase版本信息

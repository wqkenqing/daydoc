## 使用tmux

mac os 通过 brew install tmux

```
tmux 接收快捷键的指令是 ^ + B 即mac中的control + B
```

窗格的操作

```
这些操作都是通过 ^+b 来接收

```

| 操作符 | 作用     |
| :------------- | :------------- |
| %      |     左右创建两个窗格  |
| ''    |     左右创建两个窗格  |
| x    |    关闭当前窗格  |
| {    |    前移当前窗格  |
| }    |    后移当前窗格  |
| ;    |    选择上次用的窗格  |
| o    |    选择下一个窗格  |
| space   |    切换窗格布局  |
| z   |     放大窗格 |
| q   |    显示序号 |

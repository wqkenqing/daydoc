```
repeat 10 times
    tell application "System Events"
        -- 模拟在1234, 837位置的鼠标点击
        do shell script "/opt/homebrew/bin/cliclick c:1234,837"
        delay 0.5 -- 适当的延时以确保点击生效

        -- 模拟粘贴操作
        keystroke "v" using command down
        delay 0.5 -- 等待粘贴完成

        -- 模拟按回车键
        key code 36
    end tell

    delay 2 -- 每次循环之间等待2秒
end repeat

```

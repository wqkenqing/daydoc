解题:

这个题这里最大的收获应该是如何格式化文本，还有就是取小数点

格式化文本主要是str.format的应用

取小数点有好几种方法，参考[实用方法集](../python一些实用方法集.md)

我开始用到的是round(a,n) 

```
#!/usr/bin/python3


#改变标准输出的默认编码
# sys.stdout=io.TextIOWrapper(sys.stdout.buffer,encoding='utf8')

def check():
	res=input()
	val=res[0:len(res)-1]
	tag=res[len(res)-1:]
	result=cal(val,tag)
	print(result)

def cal(val,tag):
	val=float(val)
	res=0;
	if tag=="F" or tag=="f":
		res = (val - 32) / 1.8
		return "{:.2f}C".format(res)
	elif tag=='C' or tag=="c":
		res = val * 1.8 + 32	
		return"{:.2f}F".format(res)
	else:
		return "输入格式错误"
		
check()
```

## elasticsearch

思路：先建立其它名称的索引，导入新数据。再通过别名等形式将索引进行更改操作

```
POST _reindex
{
  "source": {
    "index": "hkem_electricity_data"
  },
  "dest": {
    "index": "hkem_electricity_data_new"
  }
}
```



```
POST _reindex
{
  "source": {
    "index": "hkem_electricity_device_status"
  },
  "dest": {
    "index": "hkem_electricity_device_status_new"
  }
}
```
